A minimal React app to help users find EU officials and generate paste-ready advocacy emails.

## Features

- Select country (EU only, MVP)
- Choose relevant officials (curated, official contacts only)
- Generate email drafts in 3 tones with policy asks
- Safety guardrails and disclaimers

## How to use

1. Clone/download this repo
2. Run `npm install` in `contact-finder-app`
3. Run `npm start` to develop locally
4. Deploy to GitHub Pages or similar for free hosting

## Data

- All contact and template data is in `src/data/`

---

**This tool does not send emails or store personal data. Use respectfully and lawfully.**
